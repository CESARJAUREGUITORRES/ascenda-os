📋 Cómo Funciona
1
Login básico (actual): Usuario ingresa con nombre + contraseña. Sin verificación adicional.
2
+ Verificación Email (2FA): Después del login, se envía código de 6 dígitos al email del usuario. Debe ingresarlo para acceder. Usa Resend para enviar el código.
3
+ Google Auth: Botón "Ingresar con Google" que usa Supabase Auth. El usuario no necesita contraseña, solo su cuenta Google corporativa.
4
+ CAPTCHA: Cloudflare Turnstile verifica que es humano antes del login. Invisible para el usuario, bloquea bots automáticamente.
5
+ Supabase Auth (futuro): Migración completa a Supabase Auth con JWT tokens, refresh tokens, Magic Links, y políticas RLS por usuario.
